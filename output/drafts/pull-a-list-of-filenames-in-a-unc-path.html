Pull a list of filenames in a UNC path

  • To get the text list of files in a directory, you can normally do: dir > filenames.txt
  • However, the Windows CLI can't use a UNC path as the current dir, so you need to use the Powershell CLI.
  • Once in the CLI, 'cd' to the UNC path and dir > filenames.txt will work
  • 'dir' returns a table of info along with the filenames; to just get a straight list of filenames, use Get-ChildItem -Name > C:\your\filename\here.txt}
