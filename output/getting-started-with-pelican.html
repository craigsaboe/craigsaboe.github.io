Getting Started with Pelican

Permalink: 2021-08-30 10:48:00-04:00 by Craig Saboe in misc

Code

pip install pelican
pip install markdown (*.md pages will just be ignored without this manually installed)

(Write some compelling content)

pelican content
pelican --listen
git push \master
git merge \master >> \release

Docs

https://docs.getpelican.com/en/latest/
